# Parabase
An online step sequencer built on Scratch.
Create simple melodies with built-in effects like `detune` and `release`.

> You can also export your melodies as `.mid` files using [ParaConverter](https://lukeanimationss.github.io/parabase#paraconverter).

## Usage
1. Open [Parabase](https://scratch.mit.edu/projects/1291422756/)
2. Build your melody on the grid
3. Save the melody by clicking the save button
